# Zsh-pipr

Zsh-pipr is a tiny plugin that integrates pipr with zsh. Whilst there is a zsh
script provided directly within the pipr repo, this breaks syntax highlighting
and, moreover, is not easy to install with plugin managers.

# Usage

Simply hit ctrl+p to open pipr with the current contents of the commandline.
When you wish to leave pipr, hit ctrl+c and you'll be returned to the
commandline with any extra content you added whilst using pipr.
